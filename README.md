# קורס תקשורת זוגית

דף נחיתה לקורס תקשורת זוגית מאת מיכל. אתר מודרני בעברית המציג את תוכן הקורס, מחיר, ויתרונות.

## תצוגה חיה

ניתן לצפות בדף הנחיתה בכתובת: [קורס תקשורת זוגית](https://couples-communication-course.vercel.app/)

## תכונות

- עיצוב מודרני ומקצועי
- התאמה למכשירים ניידים (רספונסיבי)
- תצוגת מחיר ברורה
- מידע על תוכן הקורס
- טופס להצטרפות
- עדויות ממשתתפים

## טכנולוגיות

- Next.js
- React
- TypeScript
- Tailwind CSS

## תחזוקה

פרויקט זה פותח על ידי מיכל ועבר שיפורים עיצוביים לשיפור חווית המשתמש והמראה המקצועי.
